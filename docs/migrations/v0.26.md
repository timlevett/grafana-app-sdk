# v0.26 → v0.28: Kind Registry Import Path

*Upgrading from*: v0.26.x

## What changed

In v0.28, the kind registry package was moved from `grafana-app-sdk/resource/registry` to
`grafana-app-sdk/kindsys/registry`. The old path is removed in v0.28.

## Automated migration

The `update-kind-registry-import` codemod handles this change automatically:

```bash
grafana-app-sdk migrate --from v0.26.0 --to v0.28.0
```

Or use `--dry-run` to preview changes first:

```bash
grafana-app-sdk migrate --from v0.26.0 --to v0.28.0 --dry-run
```

## Manual steps

None required if the codemod is applied.

## Before and after

### Before

```go
import (
    "github.com/grafana/grafana-app-sdk/resource/registry"
)
```

### After

```go
import (
    "github.com/grafana/grafana-app-sdk/kindsys/registry"
)
```

## Deprecation lint

You can detect this usage before upgrading:

```bash
grafana-app-sdk lint --deprecations
```

The scanner will report file:line locations for the old import path.
